# react-redux-saga-antd-starter

A React + Redux + Redux Actions + Redux Saga + Ant-Design frontend boilerplate.

- [x] Ant-Design
- [x] Babel
- [x] ESLint (Airbnb)
- [x] LESS
- [x] PostCSS
- [x] React
- [x] React Router
- [x] React Router Redux
- [x] Redux
- [x] Redux Actions
- [x] Redux Saga

```
npm run start
```
Open `http://localhost:3000` in your browser.
